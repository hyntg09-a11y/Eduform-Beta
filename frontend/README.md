# EDUFORM — Plataforma de Orientación Vocacional

EDUFORM es una plataforma web desarrollada con React, Node.js y MySQL que permite a los estudiantes realizar un test vocacional, recibir recomendaciones personalizadas y visualizar sus resultados mediante un dashboard interactivo.

---

## Tecnologías utilizadas

### Frontend
- React + Vite
- CSS personalizado
- Recharts (gráficos)

### Backend
- Node.js
- Express
- MySQL
- JWT (autenticación)
- bcrypt (seguridad)

---

## Requisitos

Antes de ejecutar el proyecto debes instalar:

- Node.js (versión LTS)
- MySQL o XAMPP
